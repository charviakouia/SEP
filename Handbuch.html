Anmeldung

Dies ist die Startseite des Bibliothekssystems. Sie sehen diese Seite nur, wenn Sie aktuell nicht in unserem System angemeldet ist.

Anonymer Nutzer:

geschlossener Modus
Wenn Sie keinen registrierten Account besitzen und einen benötigen, navigieren Sie auf der Randleiste zum Knopf "Registrieren". Wenn Sie bereits einen registrierten Account besitzen, tragen Sie bitte Ihre Accountdaten in die dafür vorgesehenen Felder ein, um sich anzumelden. Beachten Sie die eventuell auftretenden Meldungen neben den Formularfeldern, diese informieren Sie über fehlerhafte Eingaben. Nach erfolgreicher Anmeldung gelangen Sie zu Ihrer Profilseite.

Navigation

Anonymer Nutzer

geschlossener Modus
Durch einen Klick auf "Registrieren" werden Sie zu einer Seite weitergeleitet, auf der ein neuer Account im System angelegt werden kann. Wenn Sie sich mit Ihren Accountdaten im System anmelden wollen, leitet Sie der "Anmelden"-Knopf zur entsprechenden Seite weiter. Unter "Impressum" finden Sie rechtliche Rahmenbedingungen zur Institution und Benutzung der Bibliothek. Um genaueres über die von der Webseite gesammelten Daten und deren Verarbeitung zu erfahren, klicken Sie auf "Datenschutzerklärung". Weitere Möglichkeiten zur Kontaktaufnahme mit der Bibliotheksverwaltung finden Sie unter "Kontakt".
offener Modus
Sollten Sie sich einen Überblick über die angebotenen Medien verschaffen wollen, navigieren Sie durch einen Klick auf "Stöbern" zur Übersicht über die Medienkategorien. Wenn Sie eine Schnellsuche durch das Bibliotheksinventar starten wollen, tippen Sie ihr Stichwort in das Suchfeld und klicken Sie auf "Suche". Eine feinere Suchfunktion mit mehr Optionen steht Ihnen auf der erweiterten Suchseite zur Verfügung, welche mit einem Klick auf "erweiterte Suche" erreicht werden kann.

Registrierter Nutzer

Um Ihre Anmeldeinformationen zu verändern, gelangen Sie via "Profil" auf eine Seite mit Ihren Accountdetails.

Bibliotheksmitarbeiter

Eine Übersicht über alle aktuell zur Abholung markierten Medienexemplare finden Sie durch einen Klick auf "Abholung". Wenn Sie ein am Schalter ausgeliehenes Medienexemplar im System als ausgeliehen markieren wollen, gehen Sie über "Direktausleihe" zur entsprechenden Seite.

Administrator

Durch einen Klick auf "Verwaltung" gelangen Sie zu den Anwendungseinstellungen und der Nutzerverwaltung. Wenn Sie Impressum, Kontaktinformationen oder Datenschutzerklärung editieren wollen, klicken Sie auf den entsprechenden Reiter.

Registrierung

Hier können Sie einen neuen Account im System registrieren, um den vollen Funktionsumfang der Bibliothek nutzen zu können.

Anonymer Nutzer

geschlossener Modus
Tragen Sie bitte eine gültige E-Mail-Adresse und Ihr gewünschtes Passwort in die dafür vorgesehen Formularfelder ein (das Passwort muss bestätigt werden um Rechtschreibfehlern vorzubeugen). Beachten Sie die auftauchenden Meldungen zu fehlerhaften oder abgelehnten Eingaben neben den Formularfeldern. Die Verwaltung der Bibliothek bestimmt die Menge der zugelassenen E-Mail-Domänen. Sollte Ihre private E-Mail-Adresse abgelehnt werden, versuchen Sie es bitte mit der Adresse Ihrer Institution (Universität, Unternehmen, etc.). Die Registrierung erfolgt über die angegebene E-Mail-Adresse, zu der ein nur für Sie temporär gültiger Registrierungslink geschickt wird. Sie müssen also zeitnah auf das angegebene Konto zugreifen können, klicken Sie dann auf den Link in der E-Mail, um die Registrierung abzuschließen. Wenn Sie von Ihrer Institution auf diese Bibliothek verwiesen wurden und Probleme beim Registrieren haben, wenden Sie sich bitte an die Bibliotheksverwaltung, um genauere Informationen über die zugelassenen E-Mail-Adressen zu erhalten.
