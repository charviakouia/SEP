Anmeldung

Dies ist die Startseite des Bibliothekssystems. Sie sehen diese Seite nur, wenn Sie aktuell nicht in unserem System angemeldet ist.

Anonymer Nutzer:

geschlossener Modus
Wenn Sie keinen registrierten Account besitzen und einen benötigen, navigieren Sie auf der Randleiste zum Knopf "Registrieren". Wenn Sie bereits einen registrierten Account besitzen, tragen Sie bitte Ihre Accountdaten in die dafür vorgesehenen Felder ein, um sich anzumelden. Beachten Sie die eventuell auftretenden Meldungen neben den Formularfeldern, diese informieren Sie über fehlerhafte Eingaben. Nach erfolgreicher Anmeldung gelangen Sie zu Ihrer Profilseite. Sollten Sie Ihre E-Mail korrekt eingetragen, aber Ihr Passwort vergessen haben, klicken Sie auf "Passwort vergessen" um es zurückzusetzen. Eine E-Mail mit Zurücksetzungslink wird dann an die E-Mail-Adresse versand, unter der Ihr Account registriert ist.

Navigation

Anonymer Nutzer

geschlossener Modus
Durch einen Klick auf "Registrieren" werden Sie zu einer Seite weitergeleitet, auf der ein neuer Account im System angelegt werden kann. Wenn Sie sich mit Ihren Accountdaten im System anmelden wollen, leitet Sie der "Anmelden"-Knopf zur entsprechenden Seite weiter. Unter "Impressum" finden Sie rechtliche Rahmenbedingungen zur Institution und Benutzung der Bibliothek. Um genaueres über die von der Webseite gesammelten Daten und deren Verarbeitung zu erfahren, klicken Sie auf "Datenschutzerklärung". Weitere Möglichkeiten zur Kontaktaufnahme mit der Bibliotheksverwaltung finden Sie unter "Kontakt".
offener Modus
Sollten Sie sich einen Überblick über die angebotenen Medien verschaffen wollen, navigieren Sie durch einen Klick auf "Stöbern" zur Übersicht über die Medienkategorien. Wenn Sie eine Schnellsuche durch das Bibliotheksinventar starten wollen, tippen Sie ihr Stichwort in das Suchfeld und klicken Sie auf "Suche". Eine feinere Suchfunktion mit mehr Optionen steht Ihnen auf der erweiterten Suchseite zur Verfügung, welche mit einem Klick auf "erweiterte Suche" erreicht werden kann.

Reg

Hier können Sie einen neuen Account im System registrieren, um den vollen Funktionsumfang der Bibliothek nutzen zu können.

Anonymer Nutzer

geschlossener Modus
Tragen Sie bitte eine gültige E-Mail-Adresse und Ihr gewünschtes Passwort in die dafür vorgesehen Formularfelder ein (das Passwort muss bestätigt werden um Rechtschreibfehlern vorzubeugen). Zur Registrierung wird außerdem Ihre Adresse im dafür vorgesehenen Formularfeld benötigt. Beachten Sie die auftauchenden Meldungen zu fehlerhaften oder abgelehnten Eingaben neben den Formularfeldern. Die Verwaltung der Bibliothek bestimmt die Menge der zugelassenen E-Mail-Domänen. Sollte Ihre private E-Mail-Adresse abgelehnt werden, versuchen Sie es bitte mit der Adresse Ihrer Institution (Universität, Unternehmen, etc.). Die Registrierung erfolgt über die angegebene E-Mail-Adresse, zu der ein nur für Sie temporär gültiger Registrierungslink geschickt wird. Sie müssen also zeitnah auf das angegebene Konto zugreifen können. Klicken Sie dann auf den Link in der E-Mail, um die Registrierung abzuschließen. Wenn Sie von Ihrer Institution auf diese Bibliothek verwiesen wurden und Probleme beim Registrieren haben, wenden Sie sich bitte an die Bibliotheksverwaltung, um genauere Informationen über die zugelassenen E-Mail-Adressen zu erhalten.

Medienattributsatz

Auf dieser Seite lässt sich die Gesamtheit der setzbaren Attribute für im System gespeicherte Medien festlegen.

Anonymer Nutzer

geschlossener Modus
offener Modus

Registrierter Nutzer

Bibliotheksmitarbeiter

Administrator

Sollten Sie ein Attribut hinzufügen, wird es zu allen bestehenden Medieninstanzen mit einem leeren Wert (Platzhalter) hinzugefügt. Sie können ein neues Attribut mit dem Typ "mehrwertig" markieren, es lassen sich dann mehrere Schlagworte darunter abspeichern, dies ist z.B. für Autoren, gedacht. Sie können ein neues Attribut mit dem Typ "Referenz" markieren, hierunter kann man dann andere im System gespeicherte Medieninstanzen verlinken. Beachten Sie bitte, dass das Löschen eines Attrbuts aus dem Attributsatz alle für dieses Attribut eingetragenen Werte in den Medieninstanzen entfernt. Sie werden über den Umfang eines solchen Eingriffes vor der Durchführung gewarnt. Medieninstanzen, die sich nur in diesem Attribut unterschieden haben und nach der Löschung Duplikate darstellen, werden automatisch fusioniert.

Medienerstellung

Auf dieser Seite lässt sich eine neue Medieninstanz erstellen.

Anonymer Nutzer

geschlossener Modus
offener Modus

Registrierter Nutzer

Bibliotheksmitarbeiter

Setzen Sie eine Kategorie für das neue Medium und befüllen Sie den Attributsatz mit Werten. Sollte das Medium sich nicht klar in eines oder mehrere der Attribute einordnen lassen, können Sie das entsprechende Feld leer lassen. Wenn Sie ein als mehrwertig markiertes Attribut befüllen, fassen Sie bitte die einzelnen Schlagworte in Anführungszeichen ("). Wenn ein Attribut als Referenz markiert wurde, ist es mit einem Link auf eine andere, im System gespeicherte, Medieninstanz zu befüllen. Beachten Sie in beiden Fällen eventuell auftretende Meldungen neben den Feldern. Die Erstellung eines neuen Mediums schließen Sie durch einen Klick auf "erstellen" ab, Sie werden dann zur Nutzeransicht dieses Mediums weitergeleitet.
